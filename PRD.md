# PRD: Kenketsu-Plus

## 概要
日本赤十字社の献血ルーム・献血バスの空き状況をリアルタイム表示し、献血の予約・記録・健康管理を一元化するWebアプリ。血液型別の在庫逼迫情報を可視化し、緊急呼びかけをプッシュ通知する。

## 背景・課題
- 献血ルームの空き状況が一元的に確認できない
- 次回献血可能日の計算が複雑（全血/成分、性別で異なる間隔制限）
- 血液型別の在庫逼迫状況がリアルタイムで届かない

## ユニークポイント
血液型別の在庫逼迫度をリアルタイム可視化し、献血間隔制限を自動計算して「いつ・どこで」献血できるかを最適提案する。

## 技術スタック
- Frontend: TypeScript / Next.js / Leaflet.js / PWA
- Backend: Go (net/http)
- Database: PostgreSQL
- Infra: GCP Cloud Run (Free tier)

## 受け入れ条件
- `make check` が通る
- 全APIにユニットテスト（カバレッジ70%以上）
- 献血間隔計算のエッジケーステスト（性別×献血種別の全組み合わせ）
- Leafletマップが正常にレンダリングされる
- プッシュ通知の購読・受信フローが動作する
